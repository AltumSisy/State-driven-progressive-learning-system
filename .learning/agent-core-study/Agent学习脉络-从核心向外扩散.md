## 下一步学习目标

### 05-tool-system

**为什么下一个是工具系统？**

你已经理解了 Agent 第三层"运行时执行介入"，但工具执行的细节还不清楚：

```
┌─────────────────────────────────────────────────────────────────┐
│                      Tool System                                │
│                                                                 │
│  • beforeToolCall → 预检                                        │
│  • toolExecution → 执行                                         │
│  • afterToolCall → 后处理                                       │
│  • parallel vs sequential → 执行模式                            │
│  • tool_execution_* 事件 → 与事件系统关联                        │
│                                                                 │
│  与 Agent 的关系：                                               │
│  • Agent 第三层 "运行时执行介入" 中的工具执行                      │
│  • Agent.state.tools → 配置组装 → agent-loop → 工具执行          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**学习目标：**

1. 工具执行的完整流程
2. parallel vs sequential 执行模式
3. beforeToolCall / afterToolCall 钩子
4. tool_execution_* 事件
